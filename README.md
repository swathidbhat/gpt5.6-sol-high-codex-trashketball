# Trashketball

A cinematic, physics-based 3D paper-ball game built with Three.js and React.

## The game

- **Level 1 — The Severed Floor:** land ten shots in a cold, fluorescent office.
- **Level 2 — The Coastal House:** continue in a luxury beach house with a woven target.
- Every clean basket earns 10 points. Reach 100 points to advance.
- Visible trajectory, chargeable throw power, sound, and responsive controls.

## Controls

- **Mouse/touch:** move to aim, hold to charge, release to throw.
- **Keyboard:** arrow keys to aim and Space to shoot.

## Local development

```bash
npm install
npm run dev
```

Build the static GitHub Pages version with:

```bash
npm run build:pages
```

Pushes to `main` deploy automatically through GitHub Actions.
